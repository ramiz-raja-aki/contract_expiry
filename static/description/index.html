How it works

Users decide whether they want contract expiry reminders from their profile settings. Once enabled, the system sends a planned activity to the responsible user when a contract is nearing its end date.

1

User preferences setup

Navigate to Settings > Users & Companies > Preferences and enable the Contract Expiry Notification option for the users who should receive alerts.

2

Automatic activity creation

The module automatically creates a planned activity on the employee contract record 15 days before the end date, making follow-up visible inside the chatter.
